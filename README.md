# API Boilerplate
This is a boilerplate project for Python REST microservice APIs. 

## Prerequisites
- python ^= 3.10
- poetry ^= 1.1.12

## Getting started

#Installation

### Locally:
Clone the repository

### Install Dependences

    poetry config virtualenvs.in-project true

    poetry install

### Enter the virtual environment

    poetry shell

## How to use

    ./build.sh

    flask run
