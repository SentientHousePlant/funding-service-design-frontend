# Funding Service Design - Iteration 1
This is the first iteration of the Funding Service Design Beta application 

## Prerequisites
- python ^= 3.10
- poetry ^= 1.1.12

## Getting started

#Installation

### Locally:
Clone the repository

### Install Dependences

    poetry config virtualenvs.in-project true

    poetry install

### Enter the virtual environment

    poetry shell

## How to use

    bash ./build.sh

    flask run
