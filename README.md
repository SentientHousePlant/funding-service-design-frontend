# Funding service design frontend.

Repo for the funding service design frontend.

Built with Flask.

## Prerequisites
- python ^= 3.10
- poetry ^= 1.1.12

## Getting started

# Installation

### Locally:
Clone the repository

### Install Dependences

    poetry config virtualenvs.in-project true

    poetry install

### Enter the virtual environment

    poetry shell

## How to use

Once in the virtual enviroment:

    ./build.sh

    flask run
